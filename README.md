## Next.js App Router Course - Starter

This repository is the starter template for the Next.js App Router Course. It contains the starting code for the dashboard application.

Recommended tutorial:
- **Next.js App Router Course** on the official Next.js website
- Learn the fundamentals of App Router, server components, data fetching, layouts, and routing
- Start here: https://nextjs.org/learn

For more information, see the [course curriculum](https://nextjs.org/learn) on the Next.js website.
